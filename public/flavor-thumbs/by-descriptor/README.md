# Descriptor Photo Drop Folder

Place per-descriptor flavor images here as:

- `<slug>.png`

Slug rules used by the app:

- Lowercase
- Remove punctuation
- Spaces become `-`

Examples:

- `black cherry` -> `black-cherry.png`
- `sea spray` -> `sea-spray.png`
- `meyer lemon` -> `meyer-lemon.png`

If a descriptor image is missing, the UI automatically falls back to category images in `/flavor-thumbs/*.svg`.
